Use this path when you want the smallest set of decisions needed to get `rbxts-tailwind` running in an app or template repository.

## Recommended Order

1. Install the package and the TypeScript transformer entrypoint.
2. Add `rbxts-tailwind/transformer` to the TypeScript plugin list.
3. Create `rbxtw.config.ts` with `defineConfig()`.
4. Write a TSX element with `className` and confirm the Luau output changes.
